① 기본 구조

런치패드는 그룹 탭 → 버튼 그리드 → 재생 3단계로 동작합니다.

영역설명
상단 탭 바그룹별 탭 + 전체 탭(모든 그룹 한 화면)
버튼 그리드재생 모드에서 클릭하면 해당 구간이 즉시 재생
컨트롤 바⏸ 일시정지 / ⏹ 정지 / 볼륨 / 📺 영상 보기
오른쪽 패널 (▶/◀)닫으면 재생 모드 / 열면 편집 모드 — 버튼 편집·파서·가져오기 등
설정 저장 위치 브라우저 localStorage에 자동 저장됩니다. config.json 파일이 있으면 그것을 우선 읽습니다. 내보내기 → JSON 파일을 config.json으로 저장하면 다음 접속부터 자동 로드됩니다.

② 재생 모드 & 편집 모드

화면 오른쪽 끝의 ▶ / ◀ 토글 버튼으로 두 모드를 전환합니다.

모드패널 상태버튼 클릭 시
재생 모드 (기본)◀ 패널 닫힘해당 구간 즉시 재생
편집 모드▶ 패널 열림편집 패널만 열림, 재생 안 됨
현재 재생 중인 음악은 유지됩니다 편집 모드로 전환해도 재생 중인 음악이 멈추지 않습니다. 경기 중에 편집하면서도 응원가가 계속 나옵니다.
편집 패널이 자동으로 열리는 경우 헤더의 + 버튼 추가나 버튼의 ✏ 아이콘을 누르면 패널이 닫혀 있어도 자동으로 열립니다.

③ 그룹 관리

탭 바 오른쪽 + 그룹 추가로 그룹을 만들 수 있습니다.

탭을 우클릭하거나 탭에 마우스를 올리면 나오는 ⋯ 을 누르면:

  • ✏ 이름 변경
  • 🔄 선수 그룹 ↔ 일반 그룹 변경 — 선수 그룹은 타순/등장곡 기능이 활성화됩니다
  • 🗑 그룹 삭제 (마지막 그룹은 삭제 불가)
전체 탭 탭 바 맨 앞의 "전체" 탭을 누르면 모든 그룹의 버튼을 한 화면에서 볼 수 있습니다.

④ 버튼 편집

편집 모드(▶ 클릭으로 패널 열기)에서 버튼을 클릭하거나, 버튼 위에 마우스를 올려 나오는 ✏ 를 누르면 오른쪽 편집 패널이 열립니다.

항목내용
이름버튼에 표시되는 이름 (최대 20자)
소속 그룹드롭다운으로 다른 그룹으로 이동 가능
YouTube URL전체 URL, youtu.be 단축 URL, 또는 Video ID 모두 가능
시작 / 종료1:30 또는 90 (초) 형식
색상12가지 색상 팔레트에서 선택
종료 시간을 비워두면? 저장 시 종료 시간은 필수입니다. 영상 끝까지 재생하려면 충분히 큰 값(예: 99:00)을 입력하세요.

⑤ 선수 그룹 & 등장곡

그룹 타입을 선수 그룹으로 설정하면 버튼에 타순 번호와 등장곡 기능이 추가됩니다.

타순 설정

  • 1~9 — 선발. 번호 오름차순으로 정렬되어 표시됩니다.
  • 0 — 벤치. 선발 아래에 별도 섹션으로 표시됩니다.

등장곡 설정

편집 패널 하단 🎵 등장곡 사용 체크박스를 켜면 등장곡 구간을 설정할 수 있습니다.

  • 등장곡 URL을 비워두면 — 응원가와 동일한 영상을 사용합니다.
  • 다른 URL을 입력하면 — 별도 영상의 구간을 등장곡으로 사용합니다.

선수 카드 클릭 방법

클릭 위치동작
카드 상단 (이름 영역)등장곡 → 응원가 자동 연결 재생
같은 영상이고 구간 간격 1초 이하이면 하나의 구간으로 합쳐서 재생
🎵 등장 버튼등장곡만 재생 → 끝나면 응원가로 자동 연결
⚾ 응원 버튼응원가만 즉시 재생
✏ 아이콘 (hover)편집 패널 열기

⑥ ✂ 구간 나누기

응원가 구간을 하나의 URL로 통으로 설정해 뒀을 때, 나누기 시점 하나만 입력하면 자동으로 등장곡/응원가로 분리됩니다.

  1. 선수 그룹 버튼의 편집 패널을 엽니다.
  2. 응원가 URL / 시작 / 종료 시간을 먼저 입력합니다.
  3. ✂ 구간 나누기 버튼을 누릅니다.
  4. 나누기 시점을 입력합니다. (예: 0:20)
  5. 적용을 누르면:
      등장곡 = 기존시작 ~ 나누기시점
      응원가 = 나누기시점 ~ 기존종료
  6. 저장을 눌러 확정합니다.
등장곡 URL 생략 시 나누기를 사용하면 응원가와 동일 영상을 쓰므로 등장곡 URL 칸을 비워두면 됩니다. 저장 시 자동으로 같은 videoId를 사용합니다.

⑦ 선택 삭제

여러 버튼을 한 번에 삭제할 때 사용합니다.

  1. 런치패드 헤더 오른쪽 ☑ 선택 버튼을 누릅니다.
  2. 삭제할 버튼들을 클릭해 선택합니다. (선택된 버튼은 빨간 테두리 + ✓ 표시)
  3. 🗑 삭제 버튼을 누르면 선택한 버튼이 모두 삭제됩니다.
  4. 취소 또는 Esc 로 선택 모드를 종료합니다.

⑧ ⚡ 타임스탬프 파서

유튜브 응원가 모음 영상의 댓글·설명란 타임스탬프를 그대로 붙여넣으면 버튼을 자동 생성합니다.

사용법

  1. 오른쪽 패널에서 ⚡ 파서 탭을 선택합니다.
  2. (선택) 영상 YouTube URL을 입력합니다 — 생성되는 모든 버튼에 일괄 적용됩니다.
  3. 버튼을 추가할 대상 그룹을 선택합니다.
  4. 타임스탬프 텍스트를 붙여넣습니다.
  5. 버튼 생성을 누릅니다.

지원 형식

0:00 이름 01:44 박찬호 02:48 - 04:14 김선빈 04:14~05:10 김도영 • 05:10 이창진 [06:09] 윤도현

중복 처리

  • 같은 이름이 이미 있으면 자동으로 덮어씁니다 (시간/URL 갱신).
  • 구간이 1초 미만인 항목은 건너뜁니다.

⑨ 재생 방식 & 팝업 폴백

인라인 재생 (기본)

페이지 내 YouTube 플레이어로 재생됩니다. 📺 영상 보기 버튼으로 영상 표시/숨기기를 전환할 수 있습니다.

팝업 폴백

일부 영상은 삽입(임베드)을 허용하지 않습니다. 이 경우 자동으로 YouTube 팝업 창이 열립니다. 버튼에 팝업재생 배지가 표시됩니다.

광고 최소화

같은 브라우저에서 YouTube에 로그인돼 있으면 (다른 탭 포함) 그 세션이 공유됩니다. YouTube Premium 구독 중이면 광고가 표시되지 않습니다. Safari는 ITP 정책으로 인해 쿠키 공유가 제한될 수 있습니다.

⑩ 가져오기 / 내보내기 / 프리셋

JSON 파일로 내보내기

헤더 내보내기 버튼 → baseball-launchpad.json 파일이 다운로드됩니다.

프리셋 불러오기

헤더 📋 기아2025 버튼을 누르면 KIA 타이거즈 2025 기본 세팅(선수 응원가·타순·상황별)을 즉시 불러옵니다. 현재 데이터가 교체되므로 확인 창이 뜹니다.

내 프리셋 만들기 presets/ 폴더에 JSON 파일을 추가하면 같은 방식으로 불러올 수 있습니다 (서버에 파일을 배포한 경우).

config.json으로 설정 고정

내보낸 파일을 config.json이라는 이름으로 저장하면, 다음 접속 시 자동으로 해당 설정을 불러옵니다.

JSON 스키마

{ "groups": [ { "id": "g1", "name": "팀 응원가", "type": "general", // "general" | "players" "gridCols": 5, "buttons": [ { "id": "b1", "label": "버튼 이름", "videoId": "YouTube Video ID", "startSeconds": 0, "endSeconds": 30, "color": "#e74c3c", // 선수 그룹 전용: "battingOrder": 3, // 1~9: 선발, 0: 벤치 "entrance": { "videoId": "같거나 다른 Video ID", "startSeconds": 0, "endSeconds": 20 } } ] } ] }

⑪ 팁 & 자주 묻는 것들

타임스탬프 파서로 선수 등록 후 등장곡 설정하는 흐름

  1. 파서로 선수응원가 구간을 먼저 일괄 등록합니다.
  2. 각 선수 버튼 ✏ 편집에서 타순을 설정하면 선발/벤치가 자동 분류됩니다.
  3. ✂ 구간 나누기로 나누기 시점을 입력하면 등장곡/응원가가 자동 분리됩니다.

선수 카드가 일반 버튼처럼 보일 때

등장곡 URL이 설정되지 않은 선수는 일반 패드로 표시됩니다. 등장곡을 설정하면 이름/등장/응원 분리 카드로 바뀝니다.

재생이 안 될 때

  • YouTube URL이 올바른지 확인하세요.
  • 해당 영상이 임베드를 허용하지 않으면 팝업 창이 자동으로 열립니다.
  • 브라우저가 팝업을 차단하는 경우 주소창 오른쪽 팝업 허용 버튼을 눌러주세요.

설정 초기화

헤더 초기화 버튼을 누르면 localStorage가 지워지고 기본 설정(기아 타이거즈 샘플)으로 돌아갑니다. config.json이 있으면 그 내용으로 복원됩니다.
